I-publish sa GitHub Pages

Sundan ang mga hakbang na ito para maging live ang Saan Napunta? — libre at libre para sa lahat.

3 Steps to Go Live
GitHub Pages · Free Forever · No Credit Card
  1. Mag-login sa GitHub at gumawa ng bagong repository.
    Pangalanan ito ng:
    saan-napunta
    I-set bilang Public at i-click ang "Create repository."
  2. I-upload ang lahat ng files sa loob ng folder na ito (index.html, manifest.json, sw.js, icons/, images/). I-click ang "Upload files" sa iyong repository.
  3. Pumunta sa Settings → Pages ng iyong repository. Sa "Branch," piliin ang main at i-click ang Save.
    Ang iyong live URL ay magiging:
    https://[iyong-username].github.io/saan-napunta
